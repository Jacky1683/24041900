Glossary

HTML:- HyperText Markup Language, the standard language used to create web pages.
CSS:- Cascading Style Sheets, used to describe the presentation of a web page, including layout, colors, and fonts.
Tag:- An HTML element used to structure the content of a webpage, enclosed in angle brackets.
Attribute:- Additional information provided in an HTML tag, typically in the form of name-value pairs (e.g., `class="myClass"`).
Semantic HTML:- HTML that is written with meaningful tags that describe the content’s role or structure.
